# Generated Agent Skills

`skills/` contains generated Codex adapters. Do not edit generated `SKILL.md`
files directly. Edit the matching source in `.claude/commands/` or
`.claude/skills/`, then run `pixi run sync-ai-commands` and
`pixi run check-ai-infra`.

Synchronization may remove only DART-owned entries recorded by its manifest;
it must preserve unrelated skills in this shared catalog.
