# DART Background Theory

This directory is the release-branch home for durable theory and research
background used by DART 6.20 maintainers and agents.

DART 6 already carries the original source PDFs:

- [`../dynamics.pdf`](../dynamics.pdf) - articulated rigid-body dynamics
  foundations.
- [`../lcp.pdf`](../lcp.pdf) - linear-complementarity contact handling
  foundations.

Keep derived notes, paper maps, and reference explanations here when they are
useful across more than one task. Preserve the PDFs as authoritative source
material unless a dedicated documentation conversion verifies the Markdown
against the original source.

## Placement

- Put release-branch theory, equations, paper summaries, and reference
  foundations here.
- Put DART-specific architecture/API decisions in `docs/design/`.
- Put roadmap priority and packet sequencing in `docs/plans/`.
- Put active task evidence and resume state in `docs/dev_tasks/`.
- Put published user guidance in `docs/readthedocs/`.

For cross-bucket rules, see
[`../information-architecture.md`](../information-architecture.md).
