Tutorials
=========

The legacy OpenSceneGraph tutorial sources were removed with the renderer
replacement. Maintained visualization coverage now lives in the DART GUI
implementation behind ``dart::gui`` and the restored GUI examples.

From a source checkout, launch the standalone editor with ``pixi run dartsim``,
or run restored example names such as ``pixi run ex atlas_simbicon``,
``pixi run ex tinkertoy``, ``pixi run ex mimic_pendulums``, and
``pixi run ex rigid_chain``.
