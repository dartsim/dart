# Superseded evidence bundle

This provenance-complete scientific-negative bundle is retained as historical
evidence for the exact source and executable identities recorded in its
`metadata.json`. It is no longer the current-source bundle.

After the run, the collision-probe source received a clang-format-only change
to its repeat-count error handling and the probe target was rebuilt. The
resulting executable remained byte-identical, and the frozen experiment
parameters, trace source, solver executable, and observed scientific result did
not change. The probe source identity did change, so a fresh successor bundle
replaces this one for current-source claims.
