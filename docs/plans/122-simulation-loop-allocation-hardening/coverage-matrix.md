# PLAN-122 Coverage Matrix

This matrix tracks DART 7 simulation-loop allocation evidence. It excludes
classic DART 6 step paths unless a path is being migrated into the DART 7
`World` pipeline.

Status meanings:

- `Closed`: final post-bake evidence exists. The measured loop starts at the
  first `World::step()` after `enterSimulationMode()` or an equivalent bake.
- `Steady-state`: a useful no-allocation regression exists, but it relies on
  unmeasured warm-up simulation steps before the allocation counter starts.
- `Open`: no complete post-bake allocation row exists yet.
- `Out of scope`: excluded by PLAN-122 scope.
